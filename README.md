# Vimal Prasath S — Portfolio

A stunning, animated React portfolio with Tailwind CSS + Framer Motion.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

## 📦 What's Inside

```
src/
├── App.jsx                  # Root component
├── index.js                 # Entry point
├── index.css                # Global styles + Tailwind
└── components/
    ├── Navbar.jsx            # Sticky nav with mobile menu
    ├── Hero.jsx              # Animated hero with particle canvas
    ├── About.jsx             # About + stats section
    ├── Skills.jsx            # Tech stack grid + marquee
    ├── Projects.jsx          # Interactive project showcase
    ├── Education.jsx         # Timeline + certifications
    ├── Contact.jsx           # Contact form + links
    └── Footer.jsx            # Footer
```

## ✨ Features

- ⚡ React 18 + Tailwind CSS
- 🎬 Framer Motion animations throughout
- 🌐 Particle canvas background in Hero
- 💡 Neon glow aesthetic (cyan / pink / green)
- 📱 Fully responsive
- 🖱️ Smooth scroll navigation
- 🔠 Typewriter animation in hero
- 🎯 Scroll-triggered section reveals
- 📊 Interactive project tabs
- 📬 Contact form

## 🎨 Theme

Dark cyberpunk aesthetic with neon accents:
- Primary: `#00f5ff` (neon cyan)
- Accent: `#ff006e` (neon pink)
- Highlight: `#39ff14` (neon green)
- Background: `#030712` (near black)

## 🔧 Customization

Edit `src/components/` files to update content.  
Colors are defined in `tailwind.config.js` and `src/index.css`.
